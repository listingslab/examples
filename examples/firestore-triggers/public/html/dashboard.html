Punt Club React

Sprint 1 Delivers a working react app which also includes the Bootstrap 4 Framework and connects to real time updates on the puntclub-staging Firebase project's Cloud Firestore. It also includes a forked, customised instance of firestore console, an Open Source development tool by listingslab whose development is not part of the punt club bill. This represents 35 hours of work.
